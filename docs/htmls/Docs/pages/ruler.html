刻度尺刻度求解

引入

import ruler from 'jsdoor/ruler/index.js';

使用

引入以后，直接使用即可：

var rulerData = ruler(maxValue, minValue, num);

maxValue表示最大值，minValue表示最小值，num表示间距个数，需要注意的是，实际的间距个数可能是 num-1 或 num 或 num+1 或 1。

例子

var rulerData = ruler(198, 4, 5);

运行后 rulerData 的值： [0, 40, 80, 120, 160, 200]。

说明

我们并不总可以获得好的结果，有时候可能存在过长的小数点等问题，你可以自行进行兼容，或者 贡献代码 帮助我们改进。
